# Landing Page Psychology Examples

Use these examples to choose a concrete output shape. Do not copy claims, metrics, logos, testimonials, or urgency unless the user provides them.

## Audit Example

```markdown
| Priority | Finding | Principle | Exact fix |
| --- | --- | --- | --- |
| High | The hero leads with a vague benefit and hides the strongest metric below the fold. | Serial position effect + anchoring bias | Move the strongest credible number into the hero subhead and repeat it near the final CTA. |
| High | The primary CTA uses the same visual treatment as secondary navigation links. | Von Restorff effect + choice reduction | Make the primary CTA the only filled button; turn secondary actions into text links or muted outline buttons. |
| Medium | The feature section asks visitors to process seven equal-weight bullets. | Cognitive load theory | Collapse the section into three core outcomes and move details into supporting copy. |
```

Add a `Recommended Page Flow` when the order is the main issue.

## Page Plan Example

```markdown
1. Hero
   Job: establish the strongest claim and primary conversion action.
   Copy direction: lead with the current pain or the biggest credible anchor.
   Proof needed: one logo row, metric, customer count, or category trust marker.
   CTA: one visually dominant action.

2. Problem/value section
   Job: make the visitor feel what the current situation is costing them.
   Copy direction: frame current loss before future gain.
   Proof needed: workflow screenshot, before/after, or concrete example.
```

## Copy Rewrite Example

```markdown
Before: Save time with automated reporting.
After: Stop losing 15 hours a week to manual reporting.

Why: The rewrite turns a generic gain into a current loss, which makes the pain easier to feel and act on.
```

## Final CTA Example

```markdown
Headline: See where your reporting time is leaking
Support: Bring one dashboard. Leave with the 3 automations that save the most hours.
CTA: Book the audit
Proof line: Used by teams reporting across finance, ops, and customer success.
```
